# Joeliscoding Tap

## How do I install these formulae?

`brew install joeliscoding/tap/<formula>`

Or `brew tap joeliscoding/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joeliscoding/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
